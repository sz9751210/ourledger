## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/your-username/ourledger.git
   ```
3. **Create a new branch** for your feature or fix:
   ```bash
   git checkout -b feature/amazing-feature
   ```
4. **Make your changes** and commit them:
   ```bash
   git commit -m 'Add some amazing feature'
   ```
5. **Push to the branch**:
   ```bash
   git push origin feature/amazing-feature
   ```
6. **Submit a Pull Request** via GitHub.

## Development Setup

Please refer to the [README.md](README.md) for installation instructions.

## Code Style

- We use **TypeScript** for the frontend. Please ensure your code is typed.
- We follow standard **ESLint** rules.
- **Prettier** is used for formatting.

## Reporting Bugs

If you find a bug, please create an issue describing:
1. Steps to reproduce.
2. Expected behavior.
3. Actual behavior.

Thanks for your help!
